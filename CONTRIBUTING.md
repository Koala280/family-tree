# Contributing

Thanks for contributing to Family Tree App.

## Development Setup

1. Fork the repository.
2. Clone your fork.
3. Install dependencies:

```bash
pnpm install
```

4. Start the dev server:

```bash
pnpm dev
```

## Before Opening a Pull Request

- Build locally:

```bash
pnpm build
```

- Keep changes focused and small.
- Include clear reproduction steps for bug fixes.
- Update docs for behavior changes.

## Pull Request Checklist

- [ ] I verified the app builds locally.
- [ ] I described what changed and why.
- [ ] I included screenshots or recordings for UI changes.
- [ ] I kept unrelated changes out of this PR.

## Reporting Bugs

Please use the GitHub bug report template and include:

- Environment (OS, browser, device)
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots or console errors when possible
